技术栈

  • Python 3.10+ + uv 包管理
  • AKShare + yfinance 数据采集
  • SQLite WAL 模式 (11表)
  • pandas + numpy 量化计算
  • DeepSeek API LLM 生成
  • Streamlit + Plotly 可视化
  • APScheduler 定时调度

数据规模

  • 2000+ A股ETF 日行情
  • 5大A股指数 + 全球指数
  • 行业板块排名 (50+)
  • 北向资金 + 主力资金
  • PE/PB 历史估值序列
  • VIX + 美债收益率曲线
  • 365天+ 历史OHLCV回填

部署选项

  • 本地运行 (零成本)
  • 云服务器 2核2G (~¥50/月)
  • 树莓派/软路由 (低功耗)
  • systemd 服务托管
  • nohup 后台运行
  • nginx + Streamlit 反向代理
  • Docker 支持 (未来)
